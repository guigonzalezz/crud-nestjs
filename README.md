<h1 align="center">basic CRUD</h1>

## Description

This project it's a basic CRUD application using pipes and interceptors from NestJS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
